# Plant Monitor - Project Dependencies
# ===================================

# 
# ESP32 Dependencies:
# - PlatformIO handles ESP32 dependencies automatically
# - See platformio.ini for ESP-IDF framework configuration
#

# - Includes Flask, SQLAlchemy, Marshmallow, and testing frameworks

# - PlatformIO CLI for ESP32 development
# - Python 3.8+ for Raspberry Pi server
